# qwen-web

Run Qwen3-0.6b locally in your browser with WebGPU. Works better on Desktop then Mobile. Zero installation, logging, or analytics.

Live at: [qwen-web.sdan.io](https://qwen-web.sdan.io)

## reqs

- node >= 18.18.0
- npm or yarn
- browser with webgpu support

## install
`npm install`

## run
`npm run dev`
